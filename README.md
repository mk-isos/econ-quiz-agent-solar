# 📘 경제금융용어 퀴즈 AI Agent (MVP)

Upstage **Solar API**를 활용해
PDF 기반 경제·금융 용어집으로부터 **퀴즈를 생성하고 정답을 판정하는 AI Agent 프로토타입**입니다.

본 프로젝트는 **Upstage Ambassador 선발 과제** 제출을 목적으로 제작된
로컬 실행 기반 **MVP 데모**입니다.

---

## 0) 주제 선정 이유

나는 **2025년 1월 1일부터 매일 아침**, 한국은행에서 발간한 **「경제금융용어 700선」**을 기반으로
하루에 하나씩 경제·금융 용어를 공부하고, 이를 정리해 **블로그에 기록**해 왔습니다.

살아가면서 언젠가는 이 책을 **처음부터 끝까지 반드시 완독해야겠다는 생각**을 갖고 있었고,
그 다짐을 실천하기 위해 매일 아침 루틴으로 용어 정리를 시작했습니다.

그 결과, **2026년 1월 기준 약 400개 이상의 용어**를 정리하여 업로드했고,
이제 남은 용어들도 꾸준히 학습해 나가고 있습니다.

다만, 용어 수가 많아질수록

* 오늘 어떤 용어를 공부할지 고르는 것 자체가 번거로워지고
* 정의를 읽는 데서 끝나 암기 위주의 학습이 되기 쉽다는 한계를 느꼈습니다.

그래서 **용어를 무작위로 하나씩 뽑아 퀴즈 형태로 학습할 수 있다면**,
보다 능동적이고 오래 기억에 남는 학습이 가능하겠다고 생각했고,
이를 직접 해결해보고자 본 프로젝트를 기획·구현하게 되었습니다.

---

## 🔍 문제 정의

경제·금융 용어는 다음과 같은 특징을 가집니다.

* 정의 문장이 길고 추상적인 경우가 많음
* 서로 유사한 개념이 많아 구분이 어려움
* 반복 학습이 없으면 쉽게 잊혀짐

특히 입문자의 경우,
**정의를 읽었을 때는 이해한 것 같지만 스스로 설명하거나 구분하지는 못하는 문제**가 자주 발생합니다.

👉 단순 암기보다, **힌트를 보고 개념을 떠올리는 학습 방식**이 필요하다고 판단했습니다.

---

## 💡 해결 방안

본 프로젝트는 다음과 같은 흐름의 **AI Agent 파이프라인**으로 문제를 해결합니다.

1. **PDF 용어집 파싱**

   * Document Parse 개념을 적용해 PDF를 텍스트로 변환
2. **용어·정의 추출**

   * Information Extract 개념을 활용하여 `{term, definition}` 형태의 문제은행(JSON) 생성
3. **퀴즈 힌트 생성 Agent**

   * Solar(Chat Completions)을 사용해 정의를 기반으로
     **정답이 노출되지 않는 힌트**를 생성
4. **정답 판정 Agent**

   * 1차: 문자열 규칙 기반 판정
   * 2차: Solar를 활용한 **의미 기반 정답 판정(Yes/No + 근거)**

---

## 🤖 AI Agent 구조

```
PDF → Text
     → Terms(JSON)
         → [Solar] 힌트 생성 Agent
         → 사용자 입력
         → 규칙 판정
         → [Solar] 의미 판정 Agent
```
